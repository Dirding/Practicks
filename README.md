# Zbus — Расписание и планирование городского транспорта

Десктопное приложение на JavaFX для просмотра расписания, планирования поездок и управления транспортной системой. Позволяет пассажирам искать маршруты, а менеджерам и администраторам управлять данными.

## Возможности

- **Каталог маршрутов** — просмотр автобусов, троллейбусов, трамваев с фильтрацией по типу.
- **Поиск и фильтрация** — поиск по номеру маршрута или названию остановки в реальном времени.
- **Планировщик поездок** — поиск оптимального пути между остановками (прямые маршруты и с пересадками).
- **Избранное** — сохранение часто используемых маршрутов для быстрого доступа.
- **Панель менеджера** — управление маршрутами, остановками, транспортными средствами и расписанием.
- **Панель администратора** — управление пользователями (блокировка, смена ролей, статистика).

## Архитектура
### Приложение построено по паттерну MVC (Model-View-Controller):

- Model — классы в пакетах Class/ (сущности БД) и Manager/ (бизнес-логика).

- View — FXML-файлы в ресурсах, описывающие графический интерфейс.

- Controller — Java-классы (например, Transport.java), обрабатывающие события UI.

## Установка и запуск

### Требования

- **JDK 21** или выше (с модулями JavaFX)
- **PostgreSQL 16** (локальный сервер или удаленный)
- **Maven** (опционально, можно использовать встроенный в IDE)
- **Git**

### Настройка базы данных

1.  Установите и запустите PostgreSQL сервер.
2.  Создайте новую базу данных с именем `Cursovay`.
3.  Выполните SQL-скрипт из папки `sql/schema.sql` (если есть).
4.  **Важно:** Проверьте/измените учетные данные в классе `com.example.prilogenie.Class.Database.java`:
   
    ```java
    private static final String URL = "jdbc:postgresql://localhost:5432/Cursovay";
    private static final String USER = "postgres";
    private static final String PASSWORD = "your_password"; // Укажите ваш пароль
    
## Запуск
    
### Клонируйте репозиторий:

git clone https://github.com/Dirding/Practicks <br>
cd zbus

### Запустите сборку и выполнение через Maven:

./mvnw clean javafx:run

### Запуск из IDE
Запустите класс com.example.prilogenie.Start непосредственно из вашей IDE (IntelliJ IDEA / Eclipse).
